POI Details

Tap on any POI while in map mode, panorama mode, or search mode to open a POI information sheet that will show:

  • Name

  • Category

  • The room it's located in (if available)

Tap on the x-button, swipe down on the sheet, or tap anywhere on the map or panorama to close it.

Tap on the sheet or swipe up to view additional information, if available:

  • Description text

  • Image(s)

  • Share button

  • Edit button (if you have the necessary rights)

  • Delete button (if you have the necessary rights)

To close the view and return to the POI sheet view, swipe down anywhere on the POI Detail view or tap on the button in the top right corner.

Description Text​

The text will show all textual information entered into the POI.
Note: There might be slight differences in how the POI is displayed in the app compared to NavVis IVION on a web browser. Known limitations include: pictures embedded in the text will be displayed in a gallery below the text description documents (such as PDFs) will not be displayed, embedded videos will not be displayed, embedded iframes will not be displayed.

Images​

Images will be displayed in a gallery below the description text. Tap on any image to view it in full-screen mode.

Zoom into the image with a pinch gesture. Tap the Done (iOS) or Back (Android) button to return to the POI detail view.

Share Button​

When you tap the share button, the app will share a deep link to that POI. Choose an app to share it with from the options shown by your device.
